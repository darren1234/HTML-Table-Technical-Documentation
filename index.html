Introduction

Tables in HTML are an integral way to communicate a group of related information. A table is an arrangement of data in rows and columns, or possibly in a more complex structure. Tables are widely used in communication, research, and data analysis.

Some of the HTML table elements we'll cover are:

  • Table Body
  • Table Header
  • Table Row
  • Table Data/Cells
  • Caption

    • Defining a Table

    In HTML a table is defined with the <table></table> tags. This encompasses the entire body of the table and the look can be modified with CSS.

    Table Header

    The table header is a name representing subsequent data below (or across from) the header. It can be used to group either columns or rows of information to show a particular relationship.

    The table header is defined with the following code: 

            <th></th>

    Table Row

    A table row is simply a way for HTML to represent a row of data within a table. It can be modified using its own CSS rules.

    The table row is defined with the following code: 

            <tr></tr>

    Table Data (cells)

    'Table data' or cells, is the representation of the contents of the table. Each cell has it's own definition and can also be modified using CSS.

    The table data is defined with the following code: 

            <td></td>

    Table Caption

    The table caption is an optional element that can give meaning to the data represented. The caption tag must be applied immediately after the table tag. As with the other elements, CSS rules can be applied.

    The table caption is defined with the following code: 

            <caption></caption>

    Table Coding Example

    The coding example below shows a basic table with headers, rows, and data: 

            <table style="width:80%; background-color: rgb(163, 178, 199);">
            <caption>Company Employees</caption>
            <tr style="text-align:left;">
                <th>Firstname</th>
                <th>Lastname</th> 
                <th>Age</th>
            </tr>
            <tr>
                <td>Jill</td>
                <td>Smith</td> 
                <td>50</td>
            </tr>
            <tr>
                <td>Eve</td>
                <td>Jackson</td> 
                <td>54</td>
            </tr>
        </table>

    The corresponding table can be seen below:

    Company Employees
    Firstname Lastname Age
    Jill Smith 50
    Eve Jackson 54
